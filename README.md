# CS2 Rust Internal Base

Легковесная и безопасная база для создания внутреннего (internal) софта под CS2 на языке Rust. Архитектура построена на принципе модульности, что позволяет обновлять SDK без изменения основной логики.

## Основные возможности (Features)

- **API Hashing** — Динамический поиск адресов системных функций. В импортах DLL не отображаются подозрительные вызовы (GetAsyncKeyState, GetModuleHandle и т.д.).
- **Junk Code & Mutation** — Использование макросов для вставки "мусорного" кода и непрозрачных предикатов. Каждая компиляция создает уникальную сигнатуру файла.
- **String Encryption** — Все критические строки (названия модулей, ошибки) зашифрованы с помощью `obfstr`, что защищает от статического анализа VAC.
- **Decoupled SDK** — Логика чита отделена от данных. Поддержка быстрой замены офсетов и схем из дамперов (например, a2x).

## Требования (Requirements)

- **Rust Toolchain**: Stable или Nightly (рекомендуется последняя версия).
- **Платформа**: Windows 10/11 x64.
- **Зависимости**: Наличие установленного LLVM (для некоторых макросов обфускации).

## Использование (How to use)

1. **Обновление SDK**:
    * Сгенерируйте свежие офсеты и схемы с помощью [cs2-dumper](https://github.com/a2x/cs2-dumper).
    * Поместите файлы c расширением .rs в `src/sdk/`.

2. **Сборка**:
    Скомпилируйте проект в режиме Release для включения всех оптимизаций и механизмов защиты:
    ```bash
    cargo build --release
    ```

3. **Инжект**:
    Используйте любой надежный x64 инжектор.
    * Рекомендуемый вариант: [AnarchyInjector](https://github.com/AnarchyLoader/AnarchyInjector).
    * *Примечание: Путь к готовой DLL будет находиться в `target/release/cs2_rust_internal.dll`.*

## Структура проекта

* `src/security` - Модуль защиты, шифрования и связи с SDK.
* `src/sdk` - Хранилище динамических данных игры (офсеты/схемы).
* `src/utils` - Общие инструменты: работа с памятью, математика (W2S), мутация кода.
* `src/features` - Логика функций (ESP, Visuals и др.).
